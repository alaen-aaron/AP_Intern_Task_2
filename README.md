# AP_Intern_Task_2

# Task 2 – Network Security & Scanning

This repository contains the implementation of **Task 2 – Network Security & Scanning** from the ApexPlanet Cybersecurity & Ethical Hacking Internship.

## Objective
The objective of this task is to understand network reconnaissance, vulnerability scanning, packet analysis, and firewall configuration.

## Tools Used
- Kali Linux
- Nmap
- Nessus Essentials
- Wireshark
- iptables
- Metasploitable2

## Lab Setup
Attacker Machine: Kali Linux  
Target Machine: Metasploitable2  
Network: Host-Only Adapter

## Activities Performed
1. Network discovery using Nmap
2. Port and service scanning
3. Vulnerability scanning using Nessus
4. Packet analysis using Wireshark
5. Firewall configuration using iptables

## Results
- Multiple open ports identified
- Vulnerabilities detected in outdated services
- FTP credentials captured using packet analysis
- Firewall rule successfully blocked port scanning

## Conclusion
This task demonstrated practical skills in network scanning, vulnerability assessment, and network traffic analysis in a controlled lab environment.
